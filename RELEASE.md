# v1.3.5 — 考勤打卡 + 保存重试 + 备份

> 2026-06-07

---

## 🆕 考勤打卡

- 新增 🕐 考勤标签页，记录每日工作时间
- 自动取整当前时间 + 预选 2 小时
- 多段时段编辑（+ 添加时段），导出时同日期自动合并
- 一键请假按钮
- 最近 3 天记录显示，全部记录二级页面
- **Excel 导出**：选择起止日期（默认 30 天），格式对齐模板，列宽/加粗/行高/对齐全部保留

## 🛡 保存重试

- 网络异常时自动重试 3 次（间隔 1s → 3s → 6s）
- 重试期间 header 渐进变红，状态栏显示 `🔄 重试中 (1/3)...`
- 3 次全失败后 header 全红 + Toast 弹窗 + 红色横幅三重提醒
- 网络恢复后自动清除错误状态

## 🔧 其他

- **备份脚本** `backup.sh`：sqlite3 .backup 正确处理 WAL，每天凌晨 3 点自动备份，保留 7 天自动轮转
- **离线模拟**：三连击版本号 → Dev Tools → 勾选 📴 模拟离线
- **联动开关**：留存页可关闭与出库的联动，关闭时需二次确认
- **出库为 0 时 + 留存**：弹确认窗，本次访问只提示一次
- **标签切换记忆**：从历史页返回自动恢复上次所在标签
